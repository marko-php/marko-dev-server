# marko/dev-server

Start your full development environment with a single command.

## Installation

```bash
composer require marko/dev-server
```

## Quick Example

```bash
marko dev:up        # Start PHP server, Docker, and frontend tools
marko dev:status    # Check what's running
marko dev:down      # Stop everything
```
